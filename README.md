# WebXR POC
## Setup
`cargo install --locked trunk`

get a WebXR Emulator browser extension

[Chrome extension](https://chrome.google.com/webstore/detail/webxr-api-emulator/mjddjgeghkdijejnciaefnkjmkafnnje)
## Run
`trunk serve`

go to localhost:8080
